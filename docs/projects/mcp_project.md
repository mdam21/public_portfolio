# Utilización de MCP para poder automatizar distintos procesos en computadores

Este proyecto lo desarrollé con el objetivo de aprender y entender cómo funcionan los MCP y cómo poder aplicarlos en un entorno real con objetivos reales. Estas herramientas nos permiten comunicarnos con herramientas externas a los LLMs para poder ralizar acciones. Estos sirven principalmente para que las IAs tengan maś herramientas y más contexto sobre lo que están trabajando.
